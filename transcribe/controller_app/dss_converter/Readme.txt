**Note: Works with DSS and DS2 files

- Run Installer
- Overwrite exe from installer with exe in folder

[Usage] :
DSSConverterCLI.exe
                -convert <filepath>
                -outputfolder <folder>
                -format <.mp3 or .wav>
                -overwrite <always/no>
                -channels <1 mono / 2 stereo>
                -samplesPerSec <sample per sec value>
                -bitsPerSample <bits per sample>

example: -channels 1 -samplesPerSec 16000 -bitsPerSample 8


Latest exe will be bundled with installer when testing complete
